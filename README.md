- React-test-renderer

## Rodar projeto

### Instalar `Libs`

```
npm i
```

### Rodar `App`

```
expo start
```

> Caso demore ou trave aperte `r` para fazer o reload


### Rodar `Tests`

```
npm run test:watch
```

> E posteriomente aperte `a` para rodar todos os testes
